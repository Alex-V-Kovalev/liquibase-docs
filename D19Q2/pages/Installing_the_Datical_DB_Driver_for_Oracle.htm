Installing the Datical DB Driver for Oracle

To use the Datical Accelerator, you must install the driver for it along with the Oracle driver. 

Installing from the Update Site

Summary: Work with the Available Software dialog to configure the Name and Location of the site to work with, then choose Datical DB Drivers > Datical Accelerator to download.

Drivers for the GUI

  1. In Datical DB, select Help > Install New Software
  2. In Work with, click Add, then fill in the name and location:
    • Name: Datical Driver Updates
    • Location: http://update.datical.com/drivers/updates
    • Click OK.
  3. In the Work with dropdown, select Datical Driver Updates (which you just added).
  4. Check Datical DB Drivers, then click Next.
  5. Check only Datical Accelerator, then click Next.
  6. Click Next and follow the remaining steps to complete the installation.

Drivers for the CLI

At the command line, run the following command:
<datical_install_dir>/repl/hammer installDrivers http://update.datical.com/drivers/updates/

Installing from a Downloaded Local Repository

Summary: Download the repository, then work with the Available Software dialog to configure it to work with the repository, then choose Datical DB Drivers > Datical Accelerator to download.

Download the Repository

Download the Composite Repo from http://software.datical.com
/COMMON/Datical_DB_Software/Datical_DB_version/DaticalDBCompositeRepo-version.zip
Choose the latest version.

Drivers for the GUI

  1. In Datical DB, select Help > Install New Software
  2. In Work with, click Add, then fill in the Name and Location:
    • Name: Datical Driver Updates Repo
    • Location: Click Archive, then choose the downloaded repo zip file.
    • Click OK.
  3. In the Work with dropdown, check Datical Driver Updates (which you just added).
  4. Check Datical DB Drivers, then click Next.
  5. Check Accelerator for Oracle, then click Next.
  6. Click Next and follow the remaining steps to complete the installation.

Drivers for the CLI

At the command line, run the following command:

  • Windows
<datical_install_dir>/repl/hammer installDrivers jar:file:/<path_to_repo>/DaticalDBCompositeRepo-version.zip!/
  • Linux (note the escaped exclamation point). 
<datical_install_dir>/repl/hammer installDrivers jar:file:/<path_to_repo>/DaticalDBCompositeRepo-version.zip\!/
