1. Run the Installer (GUI)

The Datical DB installer contains both a Graphical (GUI) and Command Line (CLI) interface that can be installed on several variants of Windows and Linux operating systems. Note: This version requires that you are running a version of Linux that has a graphical interface. Once you've downloaded the correct installer for your platform - Windows (32-bit or 64-bit) or Linux (32-bit or 64-bit) – follow the instructions below to complete the install.

  1. Double-click on the installer
  2. Install the product, taking all defaults as shown below
  3. Click Next



  4. Accept the terms of the license and click Next


     

  5. Click Next 



    Note

    If you want to install different versions of Datical DB on the same machine, you will need to append the version number to the directory name (i.e. DaticalDB_X.YY.ZZZZ where X.YY.ZZZZ is the version number).

  6. If you're installing in a new directory, then you'll see this dialog. Click OK.



  7. Click Next 



  8. If you have your license file, you can Browse to it here (if not, you can install it later). Click Next.



  9. Click Next 



  10. Select Create additional shortcuts on the desktop, if desired. Click Next.



  11. The installation is finished. Click Done.





After the install completes, Datical DB is installed in the following directories.

Windows

Install Directory:  C:\Users\<Username>\DaticalDB
Datical Default Workspace: C:\Users\<Username>\datical

Linux

Install Directory:  <HomeDirectory>/DaticalDB
Datical Default Workspace: <HomeDirectory>/datical

2. Install JDBC Drivers

Datical DB needs to have access the JDBC database driver of the specific database you are planning to manage with Datical DB.

Note

Use only the drivers provided by Datical. 
  1. Start Datical DB from the Windows Start Menu
  2. (optional) Click the Cancel button if you see the New Deployment Project dialog

     

  3. Go to the menu and select Help -> Install New Software…



  4. In the Work with section, type http://update.datical.com/drivers/updates in the drop-down, then press Enter to show the list of drivers.



  5. Select the Datical DB Drivers checkbox, then click Next. 



  6. Click Next



    Note

    MySQL drivers need to be downloaded and installed manually. See Method 2 below.

  7. Select I accept the terms of the license agreements, then click Finish.



  8. Follow the instructions on any screens that appear. If you see a Security Warning window, click OK.



  9. Once the drivers are installed, you will be prompted to restart Datical DB.  Click Restart Now.

3. Install Source Control Management (SCM) Support

Install the plugins as needed.

  • Subversion
  • Git

Install the Subversion Plugin

  1. Start the Datical DB GUI
  2. Select Help > Install New Software.



  3. In the Work with section, click the Add button in the drop-down



  4. Add the download site to the Eclipse downloader, then wait for the list of installations. 

    CollabNet Eclipse Integrations
http://downloads.open.collab.net/eclipse/update-site/e3.7







     

  5. Open the Subclipse group and select the Subclipse group check box.



  6. Clear (unselect) Subclipse Integration for Mylyn 3.x (Optional)



  7. Click through Next, Next, Accept License Terms, and Finish.





  8. Let the software finish installing, then click Restart Now.



Install the Git Plugin

  1. Start the Datical DB GUI.
  2. Select Help > Install New Software.



  3. In the Work with section, select Eclipse Indigo - http://download.eclipse.org/releases/indigo/ in the drop-down.  Add it if necessary. Wait for the list of installations.



  4. Enter EGit in the filter box. When the list is updated, select Eclipse EGit, then click Next.



  5. Click through Next, Accept License Terms, Finish.



     

  6. Let the software finish installing, then click Restart Now.
