Enabling the Accelerator


You can enable the Datical Accelerator for an entire project or for individual database definitions.

Project Context

Enable the accelerator in the project settings.

  1. In the Datical DB GUI, open the project.
  2. Click the Settings tab.
  3. Click Enable Accelerator for Oracle (improves Performance) checkbox in the Snapshot & Diff Settings section.
  4. Click OK.


Database Definition Context

Enable the accelerator in the connection for the database.

  1. In the Datical DB GUI, open the project for the database.
  2. Click Edit Connection link for the database.
  3. In the Oracle Database section, selcte the Enable Accelerator for Oracle (improves Performance) checkbox.
