Configuring a Project to Use the Audit Database

Each project can be configured to record its operations and status in the Audit DB. 

  1. Open a Datical Project and select the Settings Tab



  2. Fill out the connection information to the AuditDB and click Test Connection to verify that everything was entered correctly








  3. If the database is not already setup, Datical will load the base schema. Then Datical will register the project in the Audit DB.

    You see the following message:

    The audit database AUDIT_DB was created. The project will be registered with the specified Audit DB.
