Downloading Installation Files Using the Command Line

The files that are used to install Datical Service are very large for versions 5.6.1, 5.8, and 5.9. Future versions will have smaller file sizes. In order to make it easier to download these large files, Datical has created a shell script that uses two external command line tools to download the files. 

You start the process by getting login credentials for the Datical software distribution site, https://software.datical.com/ 

Once you have the credentials, log in and use the tree on the left side of the page to navigate to Datical > FOLDERS > COMMON > Datical_Service_Software:

From there, select the download icon next to the file get-datical-service.sh to download the file to your computer. You will need to mark the file as executable. Open a terminal and change into the directory where the file was downloaded, and run 

chmod +x get-datical-service.sh

Then, still in the same terminal window, execute the command and specify the version of Datical Service you want to install:

./get-datical-service.sh 5.9

The script will first ensure that you have installed two command line utilities required. The first is curl, a tool for interacting with websites from the command line. This is typically pre-installed on most Linux distributions. The second utility is jq, a tool for parsing JSON-formatted data. This is less likely to be installed. If either tool is not installed, the script will provide instructions for installing it. 

When both utilities are installed, running the script will then prompt you for your username and password for software.datical.com. It will then begin downloading the installation files, providing progress reports during the process. Depending on your download speed, this may take several hours. 

Once you have downloaded the required files, return to the instructions Installing Datical Service on a Server or VM.
