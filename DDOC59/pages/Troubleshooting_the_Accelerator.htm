Troubleshooting the Accelerator


If the driver runs into any errors when loading, it will write the error messages out to the Datical GUI console window. Please check the console for additional information that will help to resolve the problem.
The following is a list of the most common issues that can occur when setting up this feature.
Java not installed on the database

  • Install the JavaVM as documented in the above Verify & Install Oracle Database JavaVM section.

Feature not licensed correctly

  • Install the Datical Accelerator for Oracle license

No Java permissions (but the JavaVM is installed)

  • Grant the missing permissions to the user account accessing the database.

Error installing the Stored Procedure (permissions or doesn't compile)

  • Grant the missing permissions to the user account accessing the database.
  • If the stored procedure doesn't compile, contact us at support@datical.com for assistance.
