s in JSON Format

supports JSON as a format for storing your files.

Requirements

To use JSON-based s, you must include snakeyaml-1.12.jar in your classpath in the file.

The classpath specifies the directories and .jar files to search for files and custom extension classes. If you have multiple files, they can be separated with ; on Windows or : on Linux or MacOS. Also, it is recommended to use a relative path:

classpath: ../ojdbc6-11.2.0.3.0.jar; .../ojdbc7-11.2.0.4.0

For more information, see Creating and configuring a [%=General.liquiPropFile%] file.

Limitations

None
