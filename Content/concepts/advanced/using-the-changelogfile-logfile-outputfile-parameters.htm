Using the changeLogFile, logFile, and outputFile Parameters

uses several files to store the output of your changes and database information. To create and use those files, you need logFile, outputFile, and changeLogFile parameters. You can specify these parameters in your default files, such as and pom.xml, or at runtime in the CLI.

Check the Commands documentation for more information about the parameters required for a specific command. Also, see the Creating and configuring a [%=General.liquiPropFile%] file page for detailed information on how to specify parameters in the file.
