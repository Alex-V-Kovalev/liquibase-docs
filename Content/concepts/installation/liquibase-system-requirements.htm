System Requirements

is designed to run with minimal hardware and software. We recommend that your system meet the following requirements:

Minimum Hardware Requirements

  • Disk space: 300 MB
  • Processor:
    • Intel Core i3
    • ARM processor
    • 1 CPU
  • Memory: 1 GB RAM

To increase RAM for , run the following:
Windows example: set JAVA_OPTS="-Xmx4096m" && <liquibase command>
Linux example: JAVA_OPTS="-Xmx4096m" <liquibase command>.

Minimum Software Requirements

Operating System

Any version of Windows, macOS, Unix, or Linux with a supported Java installation.

Software

  • Java 8 (We recommend Java 11 or higher).
    • If you use the Installer, Java is automatically included.
    • If you manually install , there are two options for installing Java:
      • OpenJDK by Oracle - Oracle provides the latest OpenJDK release under the open-source GPL License. This version of Java was used in the initial installation process using the installer.
      • Java by Oracle - Oracle provides the latest Java release for all operating systems.
  • Database vendor-supplied JDBC driver.
