Adding and Updating Drivers

To establish a connection between and your database, you must store the JDBC driver for your database version as a .jar file in your liquibase/lib folder.

comes with pre-installed drivers for the following databases:

  • DB2
  • Firebird
  • H2
  • HSQLDB
  • MariaDB
  • MSSQL
  • Oracle
  • PostgreSQL
  • SQLite

To use another database with , you must add the appropriate database driver to liquibase/lib. To replace a driver that’s already in liquibase/lib, delete or rename the existing JDBC file and add the new one. You can download a database’s JDBC driver from its website.

If you specify a value for the driver parameter in your file, in the CLI, or as an environment variable, make sure it matches the driver in liquibase/lib you want to use. If you use with Maven, you can specify the driver in your pom.xml file. For example:

<dependency>
    <groupId>com.oracle.database.jdbc</groupId>
    <artifactId>ojdbc8</artifactId>
    <version>13.3.0.0</version>
</dependency>
