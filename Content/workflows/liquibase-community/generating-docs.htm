Generating database documentation

Using change information stored in the s and an existing database, can generate database change documentation in a Javadoc format. To generate database documentation:

Run the db-doc command

The db-doc command is generally run through the command line only. However, there is a Maven version of this command that can be utilized as well. See the Maven db-doc topic for details.

Run the command, based on your CLI or Maven specifications. You can find more information and examples of inputs and outputs at the db-doc command or the Maven db-doc documentation.
