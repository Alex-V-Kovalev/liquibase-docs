H2 Database Tutorial for Windows

The purpose of this document is to guide you through the process of creating a new project with H2 on a Windows machine.

In this tutorial, you will generate an example project and follow the instructions to apply and learn concepts associated with creating new projects with an H2 database.

Prerequisites

  • If you have not installed the latest version of , navigate to https://download.liquibase.org/download to install the software application.
  • Ensure the liquibase.bat file’s path is set to a location in the PATH system variable.
  • Navigate to http://www.h2database.com/html/download.html and download the jdbc driver jar file for H2 database.

Tutorial

To create a project with an H2 database on your Windows machine:

  1. Create a new project folder and name it LiquibaseH2.
  2. Place the jdbc H2 jar driver file in the LiquibaseH2 project directory.
  3. In your LiquibaseH2 folder, Right-click then select New > Text Document to create an empty text file.
  4. Rename the text file to myChangeLog.xml.
