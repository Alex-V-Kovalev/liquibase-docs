What is ?

is an open-source database schema change management solution which enables you to manage revisions of your database changes easily. makes it easy for anyone involved in the application release process to:

  • Eliminate errors and delays when releasing databases.
  • Deploy and roll back changes for specific versions without needing to know what has already been deployed.
  • Deploy database and application changes together so they always stay in sync.

Frequently Asked Questions

Does work with branches?

Yes. Since each change is independent, database changes that had been made in a different branch, then merged in will be run the next time is run. You may run into a problem with the order that the statements are ran, but any issues you have can be easily solved by re-ordering the files.

 What license is released under?

is released under the Apache License, version 2.0. is released under a commercial license. The main jar file also contains commercially licensed code that is only active when a license key is provided.

 What if multiple processes or application servers attempt to migrate the database at the same time?

uses a distributed locking system to only allow one process to update the database at a time. The other processes will simply wait until the lock has been released.
