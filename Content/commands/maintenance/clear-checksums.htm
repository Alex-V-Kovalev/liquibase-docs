clear-checksums

Running the clear-checksums command

To run the clear-checksums command, specify the database driver, classpath, and URL the properties file. For more information, see Specifying Properties in a Connection Profile. You can also specify these properties in your command line.

Then run the clear-checksums command:

liquibase clear-checksums

clear-checksums global s

Definition Requirement
--url Required
--username Required*
--password Required*
