update-sql

The update-sql command is a helper command that allows you to inspect the SQL will run while using the update command.

Uses

The update-sql command is used when you want to inspect the raw SQL before running the update command, so you can correct any issues that may arise before running the command. uses the raw SQL to apply database changes you have added to the file.

Running the update-sql command

To run the update-sql command, specify the driver, classpath, and URL in the properties file. For more information, see Specifying Properties in a Connection Profile. You can also specify properties from the command line.

Run the update-sql command:

liquibase --changelog-file=sample.changelog.xml update-sql

Enter the name of the you want to use in place of sample.changelog.xml.

update-sql global attributes

Attribute Definition Requirement
--changelog-file * Required
--url Required
--username Required
--password Required
--output-file ** Optional

* checks the and any nested s for the definitions of the s to update.

* * If not specified, update SQL output goes to STDOUT.
