drop-all

Running the drop-all command

To run the drop-all command, specify the driver, classpath, and URL in the properties file. For more information, see Create and Configure a [%=General.liquiPropFile%] File. You can also specify these properties in your command line.

Then run the drop-all command:

liquibase drop-all

drop-all global s

Attribute Definition Requirement
--url Required
--username Required
--password Required

drop-all example output

When successful, the drop-all command produces the following output:

Liquibase Pro 3.9.0  licensed to Liquibase Pro Customer until Tue Sep 22 19:00:00 CDT 2020
