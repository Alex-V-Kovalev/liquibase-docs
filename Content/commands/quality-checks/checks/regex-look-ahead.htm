RegexLookAhead

The RegexLookAhead check is a pre-written Regex that automates regex work so that users do not have to hand write regex's to look forward in their database.

This check triggers when the user-supplied regular expression for a primary match is NOT followed by a user-supplied regex 'negative lookahead' match.

Uses

 

Run the RegexLookAhead check

Negative LookAhead Logic:
If regex pattern X exists then regex pattern Y must not exist (negative lookahead)
