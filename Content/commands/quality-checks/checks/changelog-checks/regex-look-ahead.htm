RegexLookAhead

The RegexLookAhead check is a pre-written Regex that automates this check so that users do not have to hand write regex's to look forward in their database.

This check triggers when the user-supplied regular expression for a primary match is followed by a user-supplied regex 'lookahead' match.

Uses

The RegexLookAhead check can be used to confirm conditions within the database. This check allows you to produce a logged record of changeset structure when certain conditions are met. When it is executed the checks message is logged to the console and logs as proof that changesets conform to specific standards.
