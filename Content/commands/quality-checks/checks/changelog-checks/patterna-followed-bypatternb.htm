PatternA_Followed_byPatternB

The PatternA_Followed_byPatternB check is a pre-written Regex quality check that allows to look for a positive pattern based on two user-provided inputs. This check is pre-configured so users do not have to hand write regex's to look forward in their .

This check triggers if the user-supplied regex pattern A is followed by the user-supplied regex pattern B.

Run the PatternA_Followed_byPatternB check

To run the PatternA_Followed_byPatternB check, ensure you have a Liquibase Pro license and the checks-scope property includes changelog. The checks-scope parameter is set in the default properties file, environment variable, or any standard method.
