updateToTagSQL <tag> command

The updateToTagSQL <tag> command is a helper command that inspects the SQL will run while using the updateToTag <tag> command.

Uses

The updateToTagSQL <tag> command is used to inspect the raw SQL before running the updateToTag <tag> command, so you can correct any issues that may arise before running the command. uses the raw SQL to apply database changes you have added to the file based on the tag specified.

Running the updateToTagSQL <tag> command

To run the updateToTagSQL <tag> command, you need to specify your driver, class path, URL, and user authentication information in your liquibase.properties file. You can also specify these in your command line.

Then run the updateToTagSQL command:

liquibase --changeLogFile=communityOnly.xml updateToTagSQL myTag

Enter the name of the and the tag that you want to use in place of "communityOnly.xml" and "myTag".
