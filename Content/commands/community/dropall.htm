dropAll command

Running the dropAll command

To run the dropAll command, you can specify the driver, classpath, and URL in your file. For more information, see Creating and configuring a [%=General.liquiPropFile%] file. You can also specify these properties in your command line.

Then run the dropAll command:

liquibase dropAll

Using the dropAll command in

If you use and want to receive the dropAll operations for your registered file, you need to specify one of the following s when running the dropAll command.

Attribute Definition Requirement
--hubConnectionId Identifies the specific Connection in which to record or extract data at . Available in your Project at https://hub.liquibase.com. Required
--hubProjectId Identifies the specific Project in which to record or extract data at . Available in your account at https://hub.liquibase.com. Required
--changeLogFile Specifies the root Required

Examples

liquibase dropAll --hubProjectId=<projectId>
liquibase dropAll --hubConnectionId=<connectionId>
liquibase --changeLogFile=changelog.xml dropAll

dropAll global s

Attribute Definition Requirement
--url Specifies the JDBC database connection URL Required
--username Specifies the database username Required
--password Specifies the database password Required
Output

When successful, the dropAll command produces the following output:
