clearCheckSums command

Running the clearCheckSums command

To run the clearCheckSums command, you need to specify your driver, class path, URL, and user authentication information in the liquibase.properties. You can also specify these properties in your command line.

Then run the clearCheckSums command:

liquibase –-changeLogFile=mainchangelog.xml clearCheckSums

Note: Enter the name of the you want to use in place of mainchangelog.xml. You can also add the changeLogFile to your liquibase.properties file.

clearCheckSums Global Attributes

Attributes Definition Requirement
--changeLogFile The root Required
--username The database username Required
--password The database password Required
--url The JDBC database connection Required
