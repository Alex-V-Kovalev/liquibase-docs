Feature Flags Overview

Liquibase Feature Flag/Management Extensions give users the ability to use some third-party feature flag tools to define and control when database changes are applied. The list of available extensions will be available on the documentation page. The extensions allow for feature flags to be used as preconditions in changelogs and changesets.

Feature Status: Labs -- user feedback requested!

Features in Labs:

  • Meet Liquibase security requirements
  • Have public user documenation available
  • Configuration and dependencies not likely to change
  • Best effort to make future versions backwards compatible
  • Recommended use in non-production environements only

    The following extensions are currently available for Liquibase Pro subscribers: launchdarkly, cloudbees-feature-management, configcat, flagsmith, gitlab-feature-flags, and growthbook. Receive a license key to start a free 30-day Liquibase Pro trial.

    The following open source extensions are currently available for all Liquibase users: ff4j, flagr, flipt. User instructions for open source extensions can be found in the READ.ME in each repository.

    Please provide feedback via the Liquibase Idea Portal.
