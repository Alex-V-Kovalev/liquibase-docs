Installing Datical DB Clients on Windows Systems

Videos

See video 14 Installing Liquibase on Windows in How Liquibase Enterprise (fka DaticalDB) Works.

Prerequisites

You have downloaded the installer for Windows systems

  • Download the following files from software.datical.com:
    • Windows installer (.exe file) - Example: DaticalDB-win-x86_64-7.16.7223.exe
    • Composite repository - Example: DaticalDBCompositeRepo-7.16.7223.zip
  • See Downloading the Datical DB Installer for more details

Verify the JRE Version on the Installation Host is 1.8

  • Run the following "java -version" command from a command prompt to check that you are running a supported version of the JRE (version 1.8.x)

    # java –version
java version "1.8.0_xx"Java(TM) SE Runtime Environment (build 1.8.0_xxx)
Java HotSpot(TM) Client VM (build 24.75-b04, mixed mode, sharing)

Installing Datical DB

  1. Double-click on the installer file (.exe).
  2. Click Next



  3. Select I accept, then click Next


     

  4. Click Next 



    Note

    If intend to install different versions of Datical DB on the same machine, append the version number to the directory name. Example: DaticalDB_2019.2.2.

  5. If you are installing in a new directory, you see a message. Click OK.



  6. You see a list of software to be installed. Leave all items checked. Click Next 



  7. License Path: If you have your license file, browse to it to install it.
    • If you do not have it, you can install it later.
  8. Click Next.



  9. You see progress bars. When they have completed, click Next 



  10. Select Create additional shortcuts on the desktop, if desired, then click Next.



  11. You see Installation Finished. Note the location of the uninstaller program. 

    Note

    To save a response file for this installation, click Generate an automatic installation script. 

    The response file (.xml) can be used to run an automated installation.  See Variation: Using the CLI and the Composite Repository to install Datical DB. 





  12. Click Done to exit the installer.

Installed Files

After the install finishes, see the following directories.

Windows

Install Directory:  C:\Users\<Username>\DaticalDB
Datical Default Workspace: C:\Users\<Username>\datical

Linux

Install Directory:  <HomeDirectory>/DaticalDB
Datical Default Workspace: <HomeDirectory>/datical
