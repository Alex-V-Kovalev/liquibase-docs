ConstraintMustExist

The ConstraintMustExist subcommand allows the user to specify a specific table by name to check that it has one or more specific constraints.

  • Subcommands require specific parameters noted in the table.
  • Customizations to checks configuration are stored in the liquibase.checks-settings file.

Run the ConstraintMustExist subcommand
