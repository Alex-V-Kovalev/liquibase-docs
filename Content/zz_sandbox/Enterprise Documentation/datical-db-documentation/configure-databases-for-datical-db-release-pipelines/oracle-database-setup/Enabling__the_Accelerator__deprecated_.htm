Enabling the Accelerator (deprecated)

You could enable the Datical Accelerator for an entire project or for individual database definitions with versions 7.15 or lower.

Deprecation of the Oracle Accelerator JDBC Driver

The accelerator/compression driver has been deprecated in version 7.16 and higher. It was previously available only with versions 7.15 and lower.

Project Setting

To enable the accelerator for all DbDefs (steps) in the project, you would enable it in the project settings for versions 7.15 or lower.

  1. In the Datical DB GUI, open the project.
  2. Click the Settings tab.
  3. Click Enable Accelerator for Oracle (improves Performance) checkbox in the Snapshot & Diff Settings section.
  4. Click OK.

DbDef Setting

To enable the accelerator only for selected DbDefs (steps), you would enable the accelerator in the database connection for the desired DbDefs with versions 7.15 or lower.

  1. In the Datical DB GUI, open the project for the database.
  2. Click Edit Connection link for the database.
  3. In the Oracle Database section, select the Enable Accelerator for Oracle (improves Performance) checkbox.
