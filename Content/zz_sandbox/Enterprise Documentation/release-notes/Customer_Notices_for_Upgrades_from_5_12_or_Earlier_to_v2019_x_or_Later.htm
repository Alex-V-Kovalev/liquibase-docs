Customer Notices for Upgrades from 5.12 or Earlier to v2019.x or Later

Important

This notice is important for anyone upgrading from older Datical 4.x or 5.x versions to the newer Datical 2019.x or 6.x versions. For example, going from Datical 5.12 to Datical 6.9.

Datical DMC

If you are using Datical Service (DMC), we strongly recommend the following:

  • Upgrade Datical Service to match the version of Datical DB. It’s important to avoid mismatches in versions as older versions of DMC are not currently guaranteed to work seamlessly with the 2019.2.2 Desktop Client and CLI. The minimum version of DMC that is compatible with DaticalDB 2019.2.2 or 6.x is Datical Service (DMC) 5.8.

  • The newest release of Datical Service/DMC is version 6.9: Datical v6.9 - Release Notes

Datical DB

If you intend to upgrade your Datical DB Desktop Client to 2019.2.2 (or higher), please note the following:

It is not possible to follow the standard upgrade process to get to the 2019.2.2 (or higher) desktop client. You need to do the following:

  1. Back up the workspace in your current installation.

  2. Perform a fresh install into a new install directory.

  3. Restore the saved workspace to the new installation.

You must follow this procedure to allow you to revert to a previous version.

For a detailed procedure, see Required Preparation for Installing the GUI Client in v2019.2 and Later

This upgrade is not compatible with older versions of the Datical desktop client. Once your workspace has been updated to 2019.2.2, you can no longer open projects with older versions of the desktop GUI client.

  • Coordinate the upgrade of the desktop client so that all users switch over to the 2019.2.2 (or higher) client. Otherwise, any users relying on the older desktop client will not be able to open and interact with projects that have been opened in a 2019.2.2 (or higher) client.

If you foresee needing to fall back to an older version of Datical DB, we strongly recommend that keep a copy of the backed-up workspace. See Required Preparation for Installing the GUI Client in v2019.2 and Later.

Changes have been made to the update site and plugins.

  • The update site (http://update.datical.com/) has been reorganized to avoid mistakes and confusion in adopting different source code plugins or performing updates. The new urls are http://update.datical.com/latest/thirdparty and http://update.datical.com/latest/gui.

  • Git plugin is now built in to the 2019.2.2 (or higher) desktop client. You do not install it separately.
