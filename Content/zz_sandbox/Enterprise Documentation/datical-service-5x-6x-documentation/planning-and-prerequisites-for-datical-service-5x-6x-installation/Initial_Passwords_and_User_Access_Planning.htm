Initial Passwords and User Access Planning

Users log in to Datical Service to use the Datical Management Console (DMC). 

Initial Passwords and Required Changes

Credentials are set for logging in to DMC and to Keycloak. 

Note

You must change the password the first time you log in. 

Logging on to DMC

DMC is the user interface for Datical Service. It has a pre-set user and passord.

You are prompted to change the password when you log in the first time. 

https:<hostname>

  • User: support@datical.com
  • Initial Password: datical

Logging on to Keycloak

Keycloak is the tool that allows you to map authentication to an existing Active Directory/LDAP service. 

You are prompted to change the password when you log in the first time. 

  1. Log on to DMC
  2. Choose Admin > Keycloak
  3. On the login page, provide the initial credentials. 
    • User: keycloak-admin
    • Initial Password: datical

User Creation and Authentication

Users must be created in Datical Service before they can access DMC. 

Create users using one of the following methods:

  • Integration with an auth service (recommended). See Configuring Datical Service Users in Keycloak. 
  • Manual creation - after you install Datical Service, you have also created an administrative DMC user in Keycloak and can log in to create users. See Datical Users.
