Database Requirements

Datical Service uses two databases.  During installation both are installed. 

  • Application database - PostgreSQL 9.6 database used as an internal component of Datical Service. 
  • Audit database - PostgreSQL 9.6 database used to record project operations. 

Configuring Datical Service to Use an Existing Audit Database

If you have an existing audit database that has been used by your Datical DB projects, configure Datical Service to use it after installation. 

An existing audit database may be on any of the following platforms: Oracle, Postgres EnterpriseDB, Microsoft SQL Server, IBM DB2.
