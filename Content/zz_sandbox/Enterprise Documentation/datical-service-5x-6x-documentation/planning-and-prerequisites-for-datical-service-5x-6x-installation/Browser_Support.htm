Browser Support

Users access the Datical Management Console (DMC) through a browser. 

DMC is built with Angular, which is built to support multiple browsers on host systems and mobile devices (phones, tablets).

See the Angular page Browser Support. 

In summary, you can access Datical Service/DMC 5.x and 6.x using the following browsers:

Browser Version Support
Chrome latest Google
Edge 2 most recent major versions Microsoft
Firefox latest Mozilla
IE (Internet Explorer) 11, 10, 9 Microsoft
IE Mobile 11 Microsoft
Safari 2 most recent major versions Apple
