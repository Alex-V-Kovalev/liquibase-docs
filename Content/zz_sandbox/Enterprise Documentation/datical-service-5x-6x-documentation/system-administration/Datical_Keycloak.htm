Datical Keycloak

Use this page to log in to Keycloak. Keycloak is used to identify an administrative user and to set up integration with an authentication service. 

The initial credentials are keycloak-admin / datical, which you must change immediately when you log in the first time. 

After you log in, you see a Datical home area in Keycloak. 

See Configuring Datical Service Users in Keycloak.
