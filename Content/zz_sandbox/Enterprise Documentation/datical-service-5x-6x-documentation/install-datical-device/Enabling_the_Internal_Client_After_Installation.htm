Enabling the Internal Client After Installation

To enable Datical operations in DMC (Forecast, Deploy) you must do the following:

  • Include the jenkins packages in your installation.
  • Enable the client after startup. 

To enable the client, run the following command:

datical-control internal-client enable

You must also install a license for the internal client. 

See Installing and Updating Licenses for Datical Service versions 5.x/6.x.
