Troubleshooting Installation Issues

Oracle: failed connection test due to timezone

If you test a connection to an Oracle database, you may run into this error if the Timezone is not set in Oracle:

To work around the issue, do the following:

  1. Create /opt/datical/docker-compose.override.yml with the contents shown below. 
  2. Run datical-services-stop.

  3. Run datical-services-start.

    version: '2.1'
services:
  reporting:
    environment:
        - DATICAL_JAVA_OPTS=-Xms200m -Xmx2000m -Duser.timezone=CDT
