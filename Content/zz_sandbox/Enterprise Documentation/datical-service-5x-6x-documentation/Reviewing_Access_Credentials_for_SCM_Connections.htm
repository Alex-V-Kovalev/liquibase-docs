Reviewing Access Credentials for SCM Connections

Credentials are stored in Keycloak as a user/password pair associated with a Credential ID, a string you create when creating the credential. 

To view access credentials and the associated IDs:

  1. Open Keycloak. In DMC, choose Settings > Keycloak. 
  2. Log in the Keycloak administrator (keycloak-admin).  
  3. Go to Credentials > System > Global credentials (unrestricted).
  4. Click Update to see the credentials associated with each ID.
