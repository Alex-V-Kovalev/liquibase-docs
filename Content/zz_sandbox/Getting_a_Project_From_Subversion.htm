Getting a Project From Subversion

This section describes how to set up a Subversion repository and check out a project. 

Subversion Repository Setup in Datical DB

You do this only once. 

Note

The following instructions use an example Subversion repository. Substitute the repository and branch for your development environment. 
  1. In the Datical DB GUI, clickOpen Perspective,then selectOther. 



  2. Select the SVN Repository Exploring, then click OK.



  3. Right-click in the blank panel, then select New > Repository Location.

     

  4. Enter the URL for the top-level SVN repository, then click Finish.



  5. The repository is now shown in the SVN Repositories tab.

Check Out Project From Subversion

  1. Navigate down the repository tree to the Datical DB project you wish to import



  2. Right-click on the repository, then select Checkout.

     

  3. Enter information as follows.
    1. Select Check out as a project in the workspace.
    2. Use the default Project Name.
    3. Click Finish.



  4. The Datical DB project is now set up in your workspace.
