TableCommentCheck

The TableCommentCheck subcommand allows the user to flag any table that does not have a comment. (Note: This is not a check for a Liquibase changelog comment.

  • Subcommands require specific parameters noted in the table.
  • Customizations to checks configuration are stored in the liquibase.checks-settings file.

Run the TableCommentCheck subcommand
