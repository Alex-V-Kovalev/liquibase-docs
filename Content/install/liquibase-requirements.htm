Requirements

is designed to run with minimal hardware and software.

Hardware

  • Disk space: 300 MB
  • Processor:
    • Intel Core i3
    • ARM processor
    • 1 CPU
  • Memory: 1 GB RAM

Software

  • Any version of Windows, macOS, or Linux/Unix with a supported Java installation.
  • Java 8, although Java 11 or later is recommended.
    • If you use the Installer, Java is included automatically.
    • If you install manually, you must also install Java, which is available from the following sources:
      • OpenJDK by Oracle – Oracle provides the latest OpenJDK release under the open-source GPL License. If you use the Installer, this version of Java is used during installation.
      • Java by Oracle – Oracle provides the latest Java release for all operating systems.
