Installation Troubleshooting

If you cannot run the liquibase --help command successfully, perform the following steps:

  1. From a command line or terminal, enter the following command to verify whether Java is installed: java -version

    2. If you receive a command not found message, visit Oracle's Java Downloads and install the latest version of Java.

  1. Verify that the system PATH variable includes the following installation directories:
  • Java (/usr/apps/Liquibase/<JDK_Directory>/bin)
  • (/path/to/Liquibase/installation)
