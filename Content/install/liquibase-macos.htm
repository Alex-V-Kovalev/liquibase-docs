Installing on macOS

Automatic

Click Install .

If the Installer does not start automatically after downloading, locate the Installer file in your Downloads directory and run it.

If you receive a malicious software alert when opening the installer, click OK and open it again. For security reasons, you might be prompted to enter your password to continue the installation process.

Manual

  1. Click Download to download the setup files.
  2. Extract the contents to a local directory.
  3. Add the installation directory to your system PATH.

    4. export PATH=$PATH:<path to liquibase installation>

  1. From a command line or terminal, type liquibase --help to verify that has been installed successfully.
