Using with InterBase

InterBase is a lightweight relational database management system with few administrative requirements. For more information, see InterBase Documentation.

Prerequisites

Install drivers

To use and InterBase, you need to have two JAR files: the JDBC driver (installed automatically) and the InterBase extension.

<dependency>
    <groupId>jdbc.interbase</groupId>
    <artifactId>interclient</artifactId>
    <version>13.2.0</version>
</dependency>

Test your connection

For and InterBase to work, you need to:

  1. Ensure your InterBase database is configured. See InterBase Quick Start for more information.
  2. Specify the database URL in the properties file:
    3. url: jdbc:interbase://localhost:3050/interbase/testdb.ib
