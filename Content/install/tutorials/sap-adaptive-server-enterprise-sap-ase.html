Using with SAP Adaptive Server Enterprise (SAP ASE)

SAP Adaptive Server Enterprise (SAP ASE) (formerly Sybase SQL Server, Sybase ASE) is a relational model database server. See SAP Adaptive Server Enterprise Documentation for more information.

Supported versions

  • 16.0.04.X
  • 16.0.03.X
  • 16.0.02.X

Verification level

Prerequisites

Install drivers

To use and SAP ASE, you need the JDBC driver JAR file. It is pre-installed in the SAP ASE database client.

Test your connection

  1. Ensure your SAP ASE database is configured. See SAP ASE Installation for more information.
    1. Select the ASE 12.0+ version.
    2. Set the select into database option.
    3. url: jdbc:sybase:Tds:127.0.0.1:5000/dbname

Rollbacks do not work on failures as Sybase does not support transactions for DDL. Additionally, foreign keys cannot be dropped, which can break the rollback or drop-all commands.
