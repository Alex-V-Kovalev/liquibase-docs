Software Configuration

The CLI ships with drivers for many popular database systems in the liquibase/internal/lib directory, including:

  • DB2
  • H2
  • HyperSQL
  • Firebird
  • MariaDB
  • MS SqlServer
  • Oracle
  • PostgreSQL
  • Snowflake
  • SQLite

If you are using a different database system or need to run a different version of the driver, you can download the appropriate driver from your vendor, and install it in one of two places:

  • The lib directory of your installation – Drivers added here will always be used by
  • A liquibase_libs directory of your current working directory – Drivers added here will only be used when you are in this working directory

If you are using any extensions, they can be added to the same directories
