Secrets Management Overview

Liquibase Secrets Management Extensions help protect credentials needed to access your database as well as any other properties you may consider sensitive. The list of available extensions will be available on the documentation page. The extensions allow for properties to point to a secret stored in an external secrets vault by a key. During runtime Liquibase will fetch the value for this secret. Temporary known limitation: only one secrets manager extension can be installed at a time.
