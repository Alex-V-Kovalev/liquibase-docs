customChange

Creates a custom class.

Uses

Although tries to provide a wide range of database s, there are times you may want to create your own custom class.

To create your own custom , simply create a class that implements the liquibase.change.custom.CustomSqlChange or liquibase.change.custom.CustomTaskChange interface <customChange> tag in your change set.

If your change can be rolled back, implement the liquibase.change.custom.CustomSqlRollback interface as well.

For a sample custom change class, see liquibase.change.custom.ExampleCustomSqlChange in the test folder

XML example

<changeSet  author="liquibase-docs"  id="customChange-example">  
    <customChange/>  
</changeSet>

YAML example

changeSet:  
  id:  customChange-example  
  author:  liquibase-docs  
  changes:  
  -  customChange:  {}

JSON example

{  "changeSet":  {  "id":  "customChange-example",  "author":  "liquibase-docs",  "changes":  [  {  "customChange":  {  }  }]  }  }
