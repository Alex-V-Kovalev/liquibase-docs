dropProcedure

Drops an existing procedure

Available s

Name Description Required for Supports
catalogName Name of the catalog asany, db2, db2z, derby, firebird, h2, hsqldb, informix, ingres, mariadb, mysql, oracle, postgresql, sybase
procedureName Name of the stored procedure to drop all all
schemaName Name of the schema all
XML example 
<changeSet  author="liquibase-docs"  id="dropProcedure-example">  
    <dropProcedure  catalogName="cat"  
            procedureName="new_customer"  
            schemaName="public"/>  
</changeSet>
 YAML example 
changeSet:  
  id:  dropProcedure-example  
  author:  liquibase-docs  
  changes:  
  -  dropProcedure:  
      catalogName:  cat  
      procedureName:  new_customer  
      schemaName:  public
 JSON example 
{
  "changeSet":{
	 "id":"dropProcedure-example",
	 "author":"liquibase-docs",
	 "changes":[
		{
		   "dropProcedure":{
		       "catalogName":"cat",
		       "procedureName":"new_customer",

   }
}
 SQL example 
DROP  PROCEDURE  cat.new_customer;
