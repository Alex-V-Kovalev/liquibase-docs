stop

Stops execution with a message. Mainly useful for debugging and stepping through a

Available attributes

Name Description Required for Supports
message Message to output when execution stops all

XML example

<changeSet  author="liquibase-docs"  id="stop-example">  
    <stop  message="What just happened???"/>  
</changeSet>

YAML example

changeSet:  
  id:  stop-example  
  author:  liquibase-docs  
  changes:  
  -  stop:  
      message:  What just happened???

JSON example

{
  "changeSet":
    {
      "id":  "stop-example",
      "author":  "liquibase-docs",
      "changes":  [{
        "stop":
          {
            "message":  "What just happened???"
          }
      }]
    }
}
