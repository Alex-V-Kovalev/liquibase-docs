dropAllForeignKeyConstraints

Drops all foreign key constraints for a table

Available attributes

Name Description Required for Supports Since
baseTableCatalogName Catalog name of the table containing columns constrained by foreign keys all 3.0
baseTableName Name of the table containing columns constrained by foreign keys all all
baseTableSchemaName Schema name of the table containing columns constrained by foreign keys all

XML example

<changeSet  author="liquibase-docs"  id="dropAllForeignKeyConstraints-example">  
    <dropAllForeignKeyConstraints  baseTableCatalogName="cat"  
            baseTableName="person"  
            baseTableSchemaName="public"/>  
</changeSet>

YAML example

changeSet:  
  id:  dropAllForeignKeyConstraints-example  
  author:  liquibase-docs  
  changes:  
  -  dropAllForeignKeyConstraints:  
      baseTableCatalogName:  cat  
      baseTableName:  person  
      baseTableSchemaName:  public

JSON example

{  "changeSet":  {  "id":  "dropAllForeignKeyConstraints-example",  "author":  "liquibase-docs",  "changes":  [  {  "dropAllForeignKeyConstraints":  {  "baseTableCatalogName":  "cat",  "baseTableName":  "person",  "baseTableSchemaName":  "public"  }  }]  }  }
