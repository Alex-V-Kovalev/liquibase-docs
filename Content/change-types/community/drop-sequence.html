dropSequence

Drops an existing sequence.

Available s

Name Description Required for Supports Since
catalogName Name of the catalog all 3.0
schemaName Name of the schema all
sequenceName Name of the sequence to drop all all
XML example 
<changeSet  author="liquibase-docs"  id="dropSequence-example">  
    <dropSequence  catalogName="cat"  
            schemaName="public"  
            sequenceName="seq_id"/>  
</changeSet>
 YAML example 
changeSet:  
  id:  dropSequence-example  
  author:  liquibase-docs  
  changes:  
  -  dropSequence:  
      catalogName:  cat  
      schemaName:  public  
      sequenceName:  seq_id
 JSON example 
{
  "changeSet":  {
    "id":  "dropSequence-example",
    "author":  "liquibase-docs",
    "changes":  [{
      "dropSequence":  {
        "catalogName":  "cat",
        "schemaName":  "public",
        "sequenceName":  "seq_id"
      }
    }]
  }
}
 SQL example 
DROP  SEQUENCE  [public].seq_id;
