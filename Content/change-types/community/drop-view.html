dropView

Drops an existing view

Available attributes

Name Description Required for Supports Since
catalogName Name of the catalog all 3.0
schemaName Name of the schema all
viewName Name of the view to drop all all

XML example

<changeSet  author="liquibase-docs"  id="dropView-example">  
    <dropView  catalogName="cat"  
            schemaName="public"  
            viewName="v_person"/>  
</changeSet>

YAML example

changeSet:  
  id:  dropView-example  
  author:  liquibase-docs  
  changes:  
  -  dropView:  
      catalogName:  cat  
      schemaName:  public  
      viewName:  v_person

JSON example

{  "changeSet":  {  "id":  "dropView-example",  "author":  "liquibase-docs",  "changes":  [  {  "dropView":  {  "catalogName":  "cat",  "schemaName":  "public",  "viewName":  "v_person"  }  }]  }  }

SQL example

DROP  VIEW  cat.v_person;
