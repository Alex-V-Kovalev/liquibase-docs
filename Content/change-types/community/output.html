output

Logs a message and continues execution.

Available attributes

Name Description Required for Supports
message Message to output all
target Target for message. Possible values: STDOUT, STDERR, FATAL, WARN, INFO, DEBUG. Default value: STDERR all

XML example

<changeSet  author="liquibase-docs"  id="output-example">  
    <output  target="STDERR">Make sure you feed the cat</output>  
</changeSet>

YAML example

changeSet:  
  id:  output-example  
  author:  liquibase-docs  
  changes:  
  -  output:  
      message:  Make sure you feed the cat  
      target:  STDERR

JSON example

{  "changeSet":  {  "id":  "output-example",  "author":  "liquibase-docs",  "changes":  [  {  "output":  {  "message":  "Make sure you feed the cat",  "target":  "STDERR"  }  }]  }  }
