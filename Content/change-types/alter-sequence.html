alterSequence

Alters properties of an existing sequence.

Available s

Name Description Required for Supports
cacheSize Change the cache size asany, db2, db2z, derby, h2, hsqldb, informix, ingres, mariadb, mssql, mysql, oracle, postgresql, sqlite, sybase
catalogName Name of the catalog asany, db2, db2z, derby, h2, hsqldb, informix, ingres, mariadb, mssql, mysql, oracle, postgresql, sqlite, sybase
cycle Can the sequence cycle when it hits the max value? asany, db2, db2z, derby, h2, hsqldb, informix, ingres, mariadb, mssql, mysql, oracle, postgresql, sqlite, sybase
dataType Data type of the sequence asany, derby, informix, postgresql
incrementBy New amount the sequence should increment by asany, db2, db2z, derby, informix, mssql, oracle, postgresql
maxValue New maximum value for the sequence asany, db2, db2z, derby, h2, informix, mssql, oracle, postgresql
minValue New minimum value for the sequence asany, db2, db2z, derby, h2, hsqldb, informix, ingres, mariadb, mssql, mysql, oracle, postgresql, sqlite, sybase
ordered Does the sequence need to be guaranteed to be generated in the order of request? asany, db2z, derby, h2, informix, oracle, postgresql
schemaName Name of the schema asany, db2, db2z, derby, h2, hsqldb, informix, ingres, mariadb, mssql, mysql, oracle, postgresql, sqlite, sybase
sequenceName Name of the sequence to alter all all
