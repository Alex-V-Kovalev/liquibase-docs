dropCheckConstraint

Drops an existing check constraint.

Available s

Name Description Required for Supports Since
catalogName Name of the catalog all 3.0
constraintName Name of check constraint to drop all all
schemaName Name of the schema all
tableName Name of the table to drop the check constraint from all all
XML example 
<changeSet  author="liquibase-docs"  id="dropCheckConstraint-example">  
    <pro:dropCheckConstraint  catalogName="cat"  
            constraintName="const_name"  
            schemaName="public"  
            tableName="person"/>  
</changeSet>
 YAML example 
changeSet:  
  id:  dropCheckConstraint-example  
  author:  liquibase-docs  
  changes:  
  -  dropCheckConstraint:  
      catalogName:  cat  
      constraintName:  const_name  
      schemaName:  public  
      tableName:  person
 JSON example 
{
  "changeSet":  {
    "id":  "dropCheckConstraint-example",
    "author":  "liquibase-docs",
    "changes":  [{
      "dropCheckConstraint":  {
        "catalogName":  "cat",
        "constraintName":  "const_name",
        "schemaName":  "public",
        "tableName":  "person"
      }
    }]
  }
}
 SQL example 
ALTER  TABLE  cat.[public].person  DROP  CONSTRAINT  const_name;
