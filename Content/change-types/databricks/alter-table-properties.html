alterTableProperties

alterTableProperties is a in the Databricks Pro extension that alters the properties of a table.

Uses

 

Run alterTableProperties

Available s

Name Type Description Requirement
tableName String to alter table properties on Required

setExtendedTableProperties

Optional.

<description>

setExtendedTableProperties has the following nested attributes:

  • tblProperties (required): description.

unsetExtendedTableProperties

Optional.

<description>

unsetExtendedTableProperties has the following nested attributes:

  • tblProperties (required): description.

Examples

  • YAML example
  • JSON example
  • XML example 
databaseChangeLog:
  - changeSet:
      id: 2
      author: your.name
      changes:
        - alterTableProperties:
            tableName: test_alter_table_properties
            setExtendedTableProperties:
              tblProperties: "'external.location'='s3://mybucket/mytable','this.is.my.key'=12,'this.is.my.key2'=true"
      rollback:
        - alterTableProperties:
            tableName: test_alter_table_properties
            unsetExtendedTablePropeties:
              tblProperties: "'external.location', 'this.is.my.key','this.is.my.key2'"
