alterCluster

alterCluster is a in the Databricks Pro extension that alters a cluster on a table.

Uses

 

Run alterCluster

Available s

Name Type Description Requirement
tableName String to alter the cluster on Required

clusterBy

Optional.

<description>

clusterBy has the following nested attributes:

  • none (required): description.

columns/column

Optional.

<description>

An array of column objects that describes columns in the table.

column has the following nested attributes:

  • name (required): description.

Examples

  • YAML example
  • JSON example
  • XML example 
databaseChangeLog:
  - changeSet:
      id: 2
      author: your.name
      changes:
        - alterCluster:
            tableName: test_table_alter_cluster
            columns:
              - column:
                  name: test_id
      rollback:
        - alterCluster:
            tableName: test_table_alter_cluster
            clusterBy:
              none: "true"
