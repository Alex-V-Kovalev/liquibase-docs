vacuumTable

vacuumTable is a in the Databricks extension that vacuums a table.

Uses

 

Run vacuumTable

Available s

Name Type Description Requirement
tableName String to optimize Required
retentionhours Integer   Optional

The DRY RUN SQL clause is not supported for the vacuumTable .

Examples

<note about rollback?>

  • YAML example
  • JSON example
  • XML example 
databaseChangeLog:
  - changeSet:
      id: 2
      author: your.name
      changes:
        - vacuumTable:
            tableName: test_vacuum_table
            retentionHours: 48
      rollback:
        - vacuumTable:
            tableName: test_vacuum_table
            retentionHours: 78
