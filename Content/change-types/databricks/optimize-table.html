optimizeTable

optimizeTable is a in the Databricks OSS extension that optimizes a table.

Uses

 

Run optimizeTable

Available s

For more information, see <API?>

Name Type Description Requirement
tableName String to optimize Required
zOrderColumns String   Required

Examples

  • YAML example
  • JSON example
  • XML example 
databaseChangeLog:
  - changeSet:
      id: 2
      author: your.name
      changes:
        - optimizeTable:
            tableName: test_optimize_table
            zOrderColumns: test_id
      rollback:
        - optimizeTable:
            tableName: test_optimize_table
            zOrderColumns: test_value
