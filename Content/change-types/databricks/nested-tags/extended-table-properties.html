extendedTableProperties

extendedTableProperties is a tag in the Databricks extension that lets you specify additional properties on a table. It is a sub-tag of the  createTable .

Uses

 

Run extendedTableProperties

extendedTableProperties is a sub-tag of the createTable . It cannot be used outside of createTable.

Available s

Name Type Description Requirement
extendedTableProperties String The table properties you want to specify. Specify properties using the format 'key=value'. Specify multiple properties in a comma-separated list. Optional

Examples

  • YAML example
  • JSON example
  • XML example 
databaseChangeLog:
  - changeSet:
      id: 1
      author: your.name
      changes:
        - createTable:
            tableName: test_table_properties
            columns:
              - column:
                  name: test_id
                  type: int
                  constraints:
                    primaryKey: true
                    nullable: false
            extendedTableProperties:
              tblProperties: 'external.location'='s3://mybucket/mytable','this.is.my.key'=12,'this.is.my.key2'=true
      rollback:
        dropTable:
          tableName: test_table_properties
