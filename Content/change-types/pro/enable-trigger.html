enableTrigger

Enables a trigger.

Available attributes

Name Description Required for Supports Since
catalogName Name of the catalog oracle, postgresql
schemaName Name of the schema all
scope all
tableName Name of the table all
triggerName all all

XML example

<changeSet  author="liquibase-docs"  id="enableTrigger-example">  
    <pro:enableTrigger  catalogName="cat"  
            schemaName="public"  
            scope="A String"  
            tableName="person"  
            triggerName="A String"/>  
</changeSet>

YAML example

changeSet:  
  id:  enableTrigger-example  
  author:  liquibase-docs  
  changes:  
  -  enableTrigger:  
      catalogName:  cat  
      schemaName:  public  
      scope:  A String  
      tableName:  person  
      triggerName:  A String

JSON example

{  "changeSet":  {  "id":  "enableTrigger-example",  "author":  "liquibase-docs",  "changes":  [  {  "enableTrigger":  {  "catalogName":  "cat",  "schemaName":  "public",  "scope":  "A String",  "tableName":  "person",  "triggerName":  "A String"  }  }]  }  }

SQL example

ENABLE  TRIGGER  [public].[A  String]    ON  cat.[public].person;
