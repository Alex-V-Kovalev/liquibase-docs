dropSynonym

Drops a synonym.

Available attributes

Name Description Required for Supports Since
objectType db2 all
private all
synonymCatalogName db2, db2z, oracle
synonymName all all
synonymSchemaName all

XML example

<changeSet  author="liquibase-docs"  id="dropSynonym-example">  
    <pro:dropSynonym  objectType="A String"  
            private="true"  
            synonymName="A String"/>  
</changeSet>

YAML example

changeSet:  
  id:  dropSynonym-example  
  author:  liquibase-docs  
  changes:  
  -  dropSynonym:  
      objectType:  A String  
      private:  true  
      synonymName:  A String

JSON example

{  "changeSet":  {  "id":  "dropSynonym-example",  "author":  "liquibase-docs",  "changes":  [  {  "dropSynonym":  {  "objectType":  "A String",  "private":  true,  "synonymName":  "A String"  }  }]  }  }

SQL example

DROP  SYNONYM  [A  String];
