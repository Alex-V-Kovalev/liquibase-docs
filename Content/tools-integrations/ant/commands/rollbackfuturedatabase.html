rollbackFutureDatabase Task

Outputs SQL for changes that have not yet been executed.

rollbackFutureDatabase task examples

<liquibase:rollbackFutureDatabase  changelog-file="/path/to/changelog.xml">  
    <liquibase:database  driver="${db.driver}"  url="${db.url}"  user="${db.user}"  password="${db.password}"/>  
</liquibase:rollbackFutureDatabase>

A basic example of rollback future task.

<liquibase:rollbackFutureDatabase  output-file="/path/to/rollback-future.sql"  changelog-file="/path/to/changelog.xml">  
    <liquibase:database  driver="${dv.driver}"  url="${db.url}"  user="${db.user}"  password="${db.password}"/>  
</liquibase:rollbackFutureDatabase>
