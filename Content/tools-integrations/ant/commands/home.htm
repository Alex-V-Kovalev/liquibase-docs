Ant Tasks

Tasks

The following tasks are available in Ant:

  • changelog-sync Ant Task
  • changelog-sync-to-tag Ant Task
  • changelog-sync-to-tag-sql Ant Task
  • db-doc Ant Task
  • diffDatabase Ant Task
  • diffDatabasetoChangelog Ant Task
  • dropAllDatabaseObjects Ant Task
  • generate-changelog Ant Task
  • mark-next-changeset-ran Ant Task
  • rollbackDatabase Ant Task
  • rollbackFutureDatabase Ant Task
  • tagDatabase Ant Task
  • updateDatabase Ant Task

Additional commands are supported by the command line that are not supported by the Ant tasks.

Migrating From Legacy Tasks

Starting with 3.3, the Ant tasks were changed, moving all of the database attributes out of the task and into its own type. The deprecated attributes will now log a warning message instructing callers of the deprecation. While backward compatibility exists, it is advised that users migrate their Ant builds to use the new tasks.

To use the new tasks, redefine the <taskdef> to use the antlib.xml file:

Here is an example of a legacy update task:

<updateDatabase changeLogFile="${db.changelog.file}"
	driver="${db.driver}"
	url="${db.url}" 
	username="${db.username}"  
	password="${db.password}" 
	promptOnNonLocalDatabase="${prompt.user.if.not.local.database}"  
	dropFirst="false"  
	classpathref="classpath"/>

Here is what it looks like migrated to the new task structure:

<liquibase:updateDatabase changeLogFile="/path/to/changelog.xml"
		dropFirst="false"
		classpathref="classpath"
		promptOnNonLocalDatabase="${prompt.user.if.not.local.database}">
	<liquibase:database driver="${db.driver}"
		url="${db.url}"
		user="${db.user}"
		password="${db.password}"/>
</liquibase:updateDatabase>
