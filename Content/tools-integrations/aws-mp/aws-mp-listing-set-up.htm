Purchase on AWS Marketplace

Follow this guide to purchase and launch from AWS Marketplace.

  1. Login to the AWS access portal.
  2. Search for .
  3. Select the 4.29.1 option.
  4. Click the View purchase options button.
  5. Select if you want to automatically renew and how many datastore targets you need.
  6. Click the Continue to Configuration button.
  7. Click the Continue to Launch button.
  8. Under the AWS License Manager section you will see the Liquibase Pro license and all of its data.

Run commands in AWS

This guide covers how to run commands against the purchased license with AWS tools such as AWS Fargate, Amazon Elastic Container Service (ECS), Amazon Elastic Kubernetes Service (EKS), AWS CodePipeline, and AWS CloudFormation.

  1. In the AWS search bar search for the tool of your choice. In this example we are using Amazon Elastic Container Service (ECS).
  2. Navigate to the Task definitions section to display all available task definitions. All tested commands are available to run as task definitions.
    1. Optional: To extend our docker image and add any other databases for a specific task definition, you must use the image url that displays the ECR repository. As of Liquibase 4.29.1 AWS does not support custom domain names for Amazon ECR.
  3. Run a task definition by selecting the radio button next to the desired definition. In this example we will use update-command.
  4. Select the Deploy button at the top of the screen.
  5. From the drop-down menu, select run task.
  6. Select the Compute options that you desire. In this example we will toggle Launch type on.
  7. AWS will bring you to the tasks in your cluster. You will be able to see the task that is running.
  8. If you are running ECS you can see the logs that display all log events on any container you have access to.
