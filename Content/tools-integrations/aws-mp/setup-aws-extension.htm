Setup the AWS Extension

Prerequisites

First, purchase a license on AWS by following our Use [%=General.Liquibase%] on AWS guide.

Step-by-step

  1. Open your Amazon S3 account.
  2. Navigate to the AWS-marketplace-listing-files bucket.
  3. Download the liquibase.docker.properties file that contains your url, username, password, and changelogFile. This is the s
  4. Navigate to AWS and then to Elastic Container Service.
  5. Open the Task definitions section from the left navigation menu.
  6. Open any task definition and notice in the command section, the liquibase.docker.properties file. This shows how the task definition can fetch the credentials specified in the liquibase.docker.properties file.   7. The liquibase.docker.properties file lives in the Liquibase owned aws-marketplace-listing-files bucket. From that file you will find the classpath specified in liquibase.docker.properties file.
