Uses

help is typically used to check the syntax of goals, their definitions, and s. The help goal displays all help information on liquibase-maven-plugin.

Maven configuration

Maven can be configured in multiple ways. One way is to define your configuration properties in your pom.xml file. To configure your pom.xml file, refer to Configuring s in your Maven POM File.

Running the help Maven goal

To run the goal, type the following in your command prompt:

mvn liquibase:help

To see s for each goal, add the -Ddetail=true :

mvn liquibase:help -Ddetail=true

If you want to see details only for one goal, add the -Dgoal=<goal name> . For example: mvn liquibase:help -Ddetail=true -Dgoal=diff.
