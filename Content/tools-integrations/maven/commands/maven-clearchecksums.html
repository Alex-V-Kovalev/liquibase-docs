Maven clear-checksums

Maven configuration

Maven can be configured in multiple ways. One way is to define your configuration properties in your pom.xml file. To configure your pom.xml file, refer to Configuring s in your Maven POM File.

Running the clear-checksums Maven goal

Running the clear-checksums goal requires a Maven project to be implemented.

To run the goal, type the following in your command prompt:

mvn liquibase:clear-checksums

Output

When successful, the clear-checksums goal produces the following output:

[INFO] Executing on Database: jdbc:oracle:thin:@support.datical.net:1521:ORCL
