Maven tag

liquibase:tag

Full name:

org.liquibase:liquibase-maven-plugin:3.8.8:tag

Description:

Writes a tag to the database.

Attributes:

  • Requires a Maven 2.0 project to be executed.
  • Requires dependency resolution of artifacts in scope: test.

Required Parameters

Name Type Since Description
liquibase.tag String - The text to write to the databasechangelog.
