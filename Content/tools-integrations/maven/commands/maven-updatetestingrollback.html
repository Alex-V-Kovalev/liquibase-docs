Maven updateTestingRollback

Maven configuration

Maven can be configured in multiple ways. One way is to define your configuration properties in your pom.xml file. To configure your pom.xml file, refer to Configuring [%=General.Liquibase%] [%=General.Param/Attr/Uppercased%]s in your Maven POM File. If your pom.xml does not reference a file, include <changeLogFile>specify your file here</changeLogFile> to the pom.xml.

Running the updateTestingRollback Maven goal

Running the updateTestingRollback goal requires a Maven project to be implemented.

To run the goal, type the following in your command prompt:

mvn liquibase:updateTestingRollback
