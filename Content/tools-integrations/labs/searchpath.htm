searchPath

searchPath is a property that is set within the file that specifically calls on the path in which the file is stored. It is a convenient way to quickly call on those files without specifying the path each time.

 

searchPath use case example:

The is stored here: “/lb-files/release1”. It is tedious to write that out each time a is specified. You can place searchPath: “/lb-files/release1” in the file
