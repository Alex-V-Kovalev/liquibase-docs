Hub

Hub provides the real-time monitoring, reports, and insights into your team's database release automation.

The Hub concepts are the following:

  • Organizations
  • [%=General.Liquibase%] Hub User
  • Projects
  • [%=General.Liquibase%] Hub Operations
  • Pipelines

Liquibase Hub functionality

The diagram represents the following actions:

  1. Joining Hub and adding API Key to the file or defaults.
  2. Registering a file.
  3. Running the commands tracked by Hub. For example, update, rollback, changelogSync, or dropAll.
  4. When successful, local DBCL table will be synchronized with Hub for Operation reports.

Need to learn how it all works? Select Getting started with [%=General.Liquibase%] Hub.
