Hub Auto-Registration and Report Links

is a free central site to help you organize and monitor your local database change activity in real-time, which increases team collaboration and deployment success. Starting with 4.3, you can use two new capabilities of :

  • Become a registered user of in one step
  • Receive a clickable link to immediately see the report of your activity

Running liquibase update or any other Hub-enabled command in the CLI (see Using Commands in [%=General.Hub%]) will give you the option to start sending data to . If you accept the option and provide your email address, then a User, Organization, Project, Target, and API key are created automatically, and your is registered to your Project. The API key and changelogID will be stored in the existing file, or if you don’t have the file, will create a new one in the current working directory.

runs your commands and sends a link for the operation report on to the console. Open the link in a browser to see the operation report.

If you just auto-registered, complete your initial sign-in using a one-time password sent to your email.

The use of capabilities is currently limited to CLI and will be extended to additional integrations in future releases.

If you use earlier versions of , refer to registerChangeLog.

If you use CI/CD tools and run in automation, the prompt is skipped and the automation continues. In this case, uses Java’s internal methods (or you can set liquibase.headless="true") to detect when running commands without interaction.
