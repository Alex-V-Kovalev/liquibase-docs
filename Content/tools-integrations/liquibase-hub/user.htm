User

At Hub, a User is a person who has an account in Hub and owns one or more API Keys.

Users can do the following:

  • Manage their profile information.
  • Register a file to synchronize it with Hub.
  • Perform local operations, which are reported to and visible in your Project’s Operations reports.
  • Create an Organization, and as the initial Organization Admin, invite other Hub users to become Organization members.
  • Create Projects in Organizations in which they are members.
  • Delete their own User account.

If you are an administrator in any Organization, you must remove yourself from the list of administrators before deleting your account.

When you join Hub, you become a User with an initial auto-created Organization for you to create Projects, store s, and more.

Users can join additional Organizations through an invitation sent by an Organization Admin to their email address. Accepting an invite is currently the only way to join additional Organizations. While an Organization Admin can invite or remove a User from their Organization, they cannot remove a User from Hub.

When Users join an Organization, they have access to all the Projects in the Organization and can provide data to any Project with a registered file. They can also see the Operations reports sent to the Organization’s Projects by all other members of the same Organization.
