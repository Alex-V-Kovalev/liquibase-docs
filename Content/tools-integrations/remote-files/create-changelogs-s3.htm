Create s with S3

To create a file with Amazon S3, follow these steps:

init project

When using init project, you cannot give attributes to s. This prevents you from using a relative path with the changelog. To solve this, use the --changelog-file property in the file to name the file from the CLI when using init project in a non-interactive way.

Configure check-settings

The check-settings file is created locally unless the absolute path is available by typing the following into the CLI:

init project --project-dir=s3://somebucket
