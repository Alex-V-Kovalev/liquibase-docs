onMissingIncludeFile

The onMissingIncludeFile is a string parameter that looks for the include file in your database. If it is set to warn and the Include file is not present, a warning message will appear that states the Include file can't be found. If it is set to fail and the Include file is not present, an error is thrown. We strongly recommend keeping the default setting foronMissingIncludeFile so it is always set to warn.

Uses

If your Include file was removed by accident Liquibase will notify you right away so you know the exact cause of the error. With onMissingIncludeFile enabled, you will be able to quickly get to the bottom of why the file is missing and how to restore it.

--on-missing-include-file
liquibase.onMissingIncludeFile
ON_MISSING_INCLUDE_FILE 		String This parameter can be set to warn you if an include file is not found. Can be set to warn or fail.
