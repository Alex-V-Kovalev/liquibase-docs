Parameters

Multiple parameters are available to help you complete the following tasks:

  • Enter properties as command-line arguments
  • Create connection profiles or specify other parameters in the properties file
  • Use the JAVA_OPTS variable to pass custom settings to the Java Virtual Machine (JVM) that runs
  • Use environment variables with Liquibase Pro to securely pass sensitive data

For more information about working with parameters, see Working with Command Parameters. To view a table of all available parameters, see Command Parameters.
