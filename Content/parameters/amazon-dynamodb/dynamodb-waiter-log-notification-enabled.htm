dynamodb-waiter-log-notification-enabled

--dynamodb-waiter-log-notification-enabled is a global Boolean parameter for the DynamoDB extension. It specifies whether the DynamoDbWaiter sends you log notifications while it polls resources in your database. The default value is true.

Uses

The --dynamodb-waiter-log-notification-enabled parameter controls whether to enable DynamoDB waiter log notifications while the waiter polls pending resources:

  • If you want to receive log notifications for waiter polling attempts, leave the parameter at its default value of true. This may be useful to analyze whether you want the waiter to poll resources in your database at a different frequency.
  • If you want to discard log notifications for waiter polling attempts, set the parameter to false. This may be useful if you already have a good understanding of how the waiter behaves in your database.
