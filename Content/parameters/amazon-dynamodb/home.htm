Parameters for Amazon DynamoDB

You can control the way behaves when running commands on your Amazon DynamoDB database by setting parameters. For example, you can use them to configure timeout values and cloud billing settings.

For a list of general parameters, see [%=General.Liquibase%] Parameters.

For more information about Amazon Dynamodb, see the getting started tutorial: Using [%=General.Liquibase%] with Amazon DynamoDB.
