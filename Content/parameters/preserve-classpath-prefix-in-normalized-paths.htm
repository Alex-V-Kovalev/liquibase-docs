preserve-classpath-prefix-in-normalized-paths

The --preserve-classpath-prefix-in-normalized-paths global parameter is a Boolean value that preserves the prefix classpath: when resolving sub-resource locations, allowing you to resolve hierarchical resources under a classpath-based root.

The default value of --preserve-classpath-prefix-in-normalized-paths is false.

Uses

 

Setting this parameter to true allows you to implement custom ResourceAccessors that work with abstract URLs without being directly tied to the file system.
