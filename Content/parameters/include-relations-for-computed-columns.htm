include-relations-for-computed-columns

--include-relations-for-computed-columns is a global Boolean parameter that specifies whether commands like diff and snapshot preserve the parent relationship for computed columns. The default value is false. This parameter is available in 4.25.0 and later.

Uses

A computed column in a table uses data from other columns to calculate its value. commands like diff and snapshot create a snapshot of your database, including any computed columns.

By default, the snapshot generates on databases like Microsoft SQL Server does not associate computed columns with the table(s) it uses to determine its values. This means that may incorrectly detect unexpected columns when you run snapshot-dependent commands.
