Parameters for Databricks

You can control the way behaves when running commands on your Databricks database by setting parameters. For example, you can use them to configure OAuth authentication.

For a list of general parameters, see [%=General.Liquibase%] Parameters.

For more information about Databricks, see Using [%=General.Liquibase%] with Databricks Platforms. For a getting started tutorial, see Using [%=General.Liquibase%] with Databricks SQL.
