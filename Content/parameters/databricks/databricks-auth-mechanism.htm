databricks-auth-mechanism

--databricks-auth-mechanism is a global parameter for the Databricks extension. It specifies the authentication mechanism for to use with Databricks. Valid values are: OAUTH. If no value is specified, tries to connect using a username and password.

Specifying this parameter is required for OAuth authentication. It is available in the Databricks extension 1.0.0 and later. For more information, see Databricks: Authenticate access to Databricks resources.
