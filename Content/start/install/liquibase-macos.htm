Installing on macOS

Automatic

Click Install .

If the Installer does not start automatically after downloading, locate the Installer file in your Downloads directory and run it.

Manual

  1. Click Download to download the installation files. For more information, see [%=General.Liquibase%] Installation Files.
  2. Extract the contents to a local directory.
  3. (Optional) Add the installation directory to your system PATH. Open the ~/.bashrc file (if you use Bash) or ~/.zshrc (if you use Zsh) and add this line:

    4. export PATH=$PATH:<path to liquibase installation>

    Restart the shell session to load the new changes.

  4. From a command line or terminal, type liquibase --version to verify that has been installed successfully.
