Amazon DynamoDB Extension Release Notes

Release notes for the extension for Amazon DynamoDB.

The release of the liquibase-amazon-dynamodb extension is independent of liquibase-core releases. Versions numbers of this extension and liquibase-core are not necessarily synchronized. See the compatibility chart for more information.

Amazon DynamoDB 1.0.0 – March 19, 2024

Initial release of the Amazon DynamoDB extension. Contains support for XML, YAML, and JSON s, as well as partial PartiQL support. Support for many features is included.
