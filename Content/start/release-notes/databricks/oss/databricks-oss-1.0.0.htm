Databricks 1.0.0 Release Notes

Databricks 1.0.0 is a major release. It was released on May 19, 2023. See GitHub: Databricks v1.0.0.

For documentation and guidance on using Liquibase with Databricks, see: Using [%=General.Liquibase%] with Databricks Platforms.

Notable Changes

  • Initial release of the Databricks Liquibase extension.
  • Passes Liquibase "Foundational" Verification.
