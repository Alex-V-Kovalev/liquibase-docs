Using with Amazon DynamoDB

Amazon DynamoDB is a fully managed NoSQL database service. You can use Amazon DynamoDB to create a database table that can store and retrieve any amount of data, and serve any level of request traffic. Amazon DynamoDB automatically spreads the data and traffic for the table over a sufficient number of servers to handle the request capacity specified by the customer and the amount of data stored. for more information, see the Amazon DynamoDB documentation page.
