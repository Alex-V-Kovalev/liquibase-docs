Databricks Pro extension permissions

To use commands and features with Databricks, you must enable the correct permissions in the Databricks Data Editor dialog.

Procedure

  1. Navigate to the Databricks Data Editor dialog.
  2. Ensure that the following privilege presets are checked:
    1. Prerequisite: 
      • USE SCHEMA
    2. Metadata: 
      • APPLY TAG
    3. Read: 
      • READ VOLUME
      • SELECT
    4. Edit: 
      • MODIFY
      • WRITE VOLUME
    5. Create: 
      • CREATE FUNCTION
      • CREATE MATERIALZED VIEW
      • CREATE MODEL
      • CREATE TABLE
      • CREATE VOLUME
