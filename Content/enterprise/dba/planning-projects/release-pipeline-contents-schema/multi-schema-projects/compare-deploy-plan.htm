Compare Deployment Steps in a Multiple Schema Deployment Plan

Comparing Deployment Steps works just as it did before, with the additional option of selecting subsets of schema to compare in a Multiple Schema Deployment Plan. Selecting subsets of schema can save time during a comparison, especially in a plan where there are many Managed Schema. To compare subsets of Managed Schema, consult the screen shot and explanation below

  1. Launch the Database Compare Wizard
  2. Configure the Comparison options as required
  3. Check Only Compare a Subset of Managed Schema in the Options section
  4. Click Configure.
  5. Use the resulting dialog to move the desired schema from the Available list to the Assigned list.
  6. Click Ok
  7. Click Finish

At this point the comparison should execute and only differences in the specified schema should be included.
