modes

The may be run in two configurations:

  • Demo mode
  • Production mode

Demo mode

Demo mode allows you to use every available feature and configuration in a safe setting without impacting your live database. The demo mode runs on an internal PostgreSQL database which has set up for you.

[%=General.DMCProductShortName%] demo mode

Production mode

Production mode allows you to configure and run all features on your live database. Currently, production mode supports PostgreSQL 10, 11, 12, or 13 databases or AWS Postgres RDS.

[%=General.DMCProductShortName%] production mode
