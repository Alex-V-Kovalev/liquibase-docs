Uninstalling the

The () allows users to uninstall the directly from the command line interface (CLI).

Uses

Using the uninstall command removes all auto-generated files in datical_service_home & anywhere on your operating system. It will also move all non-auto-generated files such as config files & log files to a backup location in datical_service_home.

Uninstalling the

To uninstall the :

  • In your command prompt, run the following command:
sudo /opt/datical/uninstall

/opt/datical is the location where was installed.
