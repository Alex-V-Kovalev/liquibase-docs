Configure your hostname

There are a variety of reasons why you may need to change or update the hostname of your web server.

Changing the hostname

To change the hostname, type the following command from a terminal logged into your host:

sudo datical-control network set-external-hostname --hostname <new-hostname>

When to update the hostname on all existing projects

It is important to note that in some cases, if you change the hostname in DMC, you will need to make the same changes to all existing projects. Learn the scenarios you may or may not need to update your datical.project file.

  1. if is in prod mode, then runs on one server and DMC DB is stored on another server.

    in this case - No, changes to the datical.project file are not needed.

  2. if is in demo mode and the hostname in this command sudo datical-control network set-external-hostname --hostname <new-hostname> has been changed to another FQDN.

    in this case - Yes, changes to datical.project file are needed.

  3. if is in demo mode and hostname is set to some value, but datical.project contains IP of the server (not a hostname), and hostname is changed but IP is not.

    in this case - No, changes to datical.project are not needed because it still communicates via IP and IP has not been changed.
