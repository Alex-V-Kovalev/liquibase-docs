Enable Display of Password on Client Connect Page in DEMO Mode

By default, the password for the embedded is not displayed for non-administrator users on the Client Connect page when the is installed in Demo Mode. The can be configured to show this password to all logged in users.

These instructions only apply when the was installed in DEMO mode. In PROD mode the password is never visible on the Client Connect page.

To enable the display of this password for non-administrators:

  1. On the system where the is installed, open the DATICAL_HOME/config/cluster.properties file in a text editor.
  2. Edit or add the datical.db.showPassword property so that it is set to true.

    3. datical.db.showPassword=true

  3. Restart datical-service using the following command:
    4. sudo service datical-service restart
  4. After the service restarts, log in to the as a non-administrator and go to the Client Connect page by clicking:

The password should be presented to the non-administrator user account.
