Configuring a new server to use an existing

If you need to move your installation to a different system for any reason, configuring the new system to use your existing requires additional steps during installation on the new system.

To configure a new installation to use an existing :

  1. Copy the files from DATICAL_HOME/config/cluster.properties on the original system and save them to /tmp on the new system.
  2. Shut down the original system.
  3. Rename the /tmp/cluster.properties file on the new system to /tmp/restore.cluster.properties.
  4. Run the installer on the new system and provide the same database connection information that was used to configure the original system.
  5. After your successful install on the new system, delete the /tmp/restore.cluster.properties file.
