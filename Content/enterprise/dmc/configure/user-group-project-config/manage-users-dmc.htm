Managing users in the

The User Settings page lists all users who can log into the (). This view is only available to users with the System Administrator role. The view allows you to view, search, and edit users that have been created in Keycloak or imported into Keycloak from your LDAP server.

See the Creating [%=General.DMCProductShortName%] users in Keycloak topic to learn how to add users to the .

Visible user information

The following information is shown for the selected user in the :

  • Full Name - First name, last name
  • Email - Email address

User settings view

The User Settings view appears after you have selected the user you want to view from the User Settings page. In this view you can elect to make the user a system administrator, and add or remove projects from their view.
