How To: Create and manage a new project with Team Foundation Server (TFS)

Step-by-step guide

Creating a new project TFS project 

        1.   Open Visual Studio, under Source Control Explorer tab,  make sure you are on the right Workspace.

                   a. In order To add a new workspace, click on the drop-down menu under Workspaces → fill in workspace name → click ok.

                   

                   

       2. Create a new team project.  Go to file → New → Team Project

               a.

               

               b. Fill in your new team project –> click next

               

      3. Map your project to the project folder path. Under Local Path you will notice that currently the link is "Not mapped".  That's because we need to map the Server to your local path.

              a. Click on → Not mapped → Fill in your path → click yes.

              

              

             b. You should be able to see the full path now.

Importing and managing a project in with the TFS plugin.

1. Adding the TFS project we have previously created in Visual Studio to .

                 a.  Open  → Open Perspective → choose Team Foundation Server Exploring → click OK

                 

                 b. Click to Team Foundation Server Explorer tab, then Click HOME  →  → Projects and My Teams →  → Connect to Team Projects

                 

                 

                 

                 c. Select your project and click → Finish

                 

                  d. Make sure that you are in the right Workspace.  To change or confirm that, Go to HOME  → click on the TFVC Pending Changes →   → Actions →  → Switch Workspaces → <your workspace>

                 

                 

                 

2. Adding and managing files to the TFS server and importing the project.

                a. To add the files (similar to "git add .") from the path to the TFS server, right click on your project → Add Items to Folder → select items → click finish

               

               

               

               b. To commit changes, enter Comment in the comment field and click the "check In button" in the "TFVC Pending Changes" Panel (Similar the git commit -m and git push all together)

              

              

              c. To import your project, click File → Import → Team → Projects from Team Foundation Server → next → select <your project> → next 

                  Under "Project Selection": select your project folder (where your datical.project resides) → click next → finish.  You should now be able to see your project under the Deployment Plans tab.

    a. Update your Workspaces.

                 tf workspaces /collection:http://localhost:8095/tfs/DefaultCollection
