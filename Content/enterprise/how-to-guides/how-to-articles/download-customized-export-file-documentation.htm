How To: Download a customized export file (HTML, XML, PDF) of confluence documentation

Note: This feature is available to the designated Customer Administrators (primary customer contact). If you are a Customer Administrator and cannot access this information, please send an email to support@liquibase.com so we can verify your permission level (user account needs to belong to the Export-only group).

  1. Login to confluence
  2. Navigate to the space that contains the document you need exported
    1. Knowledge Base
    2. Liquibase Enterprise/Datical Product Documentation
    3. Your company specific collaboration space (varies by company)
  3. Navigate to the page or section you are exporting
    1. Click the Gear Icon/Space Settings in the left icon panel
    2. Select Content Tools
    3. Select Export
    4. Select the appropriate Export Format
    5. Click Next
  4. Export the Custom Document
    1. Select the Custom Export option
    2. Click Deselect All
    3. Select the pages desired for the custom document
    4. Click Export
  5. Download the completed document
    1. Once the progress bar is 100%, click the here link under the progress bar to download the new document.
