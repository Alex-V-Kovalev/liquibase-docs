IBM UrbanCode Build Integration


Download the DaticalDB4UrbanCode Plug-in

  1. Download the DaticalDB4UrbanCode plugin. .
  2. Go to the following URL: https://github.com/IBM-UrbanCode/DaticalDB-UCD-UCB/releases
  3. In the latest release, click the .ZIP file (DaticalDB-UCD-UCB-<version>) to download it. 

Install the Plug-in 

  1. In UrbanCode Build, navigate to System > Plugins.
  2. Click Choose File.
  3. Select the downloaded DaticalDB4UrbanCode zip file.
  4. Click Submit.

Set Datical Properties

  1. Click Agents and go to the agent that will execute .
  2. Click Properties to list the agent properties.
  3. Click Add Property to enter the following properties.
    • daticalDBCmd
    • daticalDBDriversDir

Add Datical Steps to UrbanCode Build Jobs

  1. Go to a Job. 
  2. Add a Step before or after
  3. Go to the node.



  4. Add the step from Available Plugin Steps. For example, select > Check Drivers.
  5. Configure the step for your environment. The example shows the Check Drivers options.



  6. Add further steps as needed.
