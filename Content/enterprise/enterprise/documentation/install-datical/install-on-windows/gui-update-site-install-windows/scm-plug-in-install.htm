Installing SCM Plug-ins on Windows Systems

Optional Installation

The SCM Plug-ins allow you to perform source control operations from the Datical DB GUI for Subclipse/SVN/Subversion or for Team Explorer Everywhere/TFS/Team Foundation Server.

Installing them is optional. You can also choose to work directly with SCM applications. 

Note that Git functionality is already included, so you do NOT need to install any plugins if you use Git as your source control.

Prerequisites

  • You have configured the installation repositories. See Configuring Installation Repositories. 

Installing a Plug-in

  1. Start the Datical DB GUI
  2. In the Datical DB window, select Help > Install New Software. You see the Available Software window. 



  3. Click the selection arrow in the Work with field. You should see the following list of available sites:

    --All Available Sites--
Datical DB ThirdParty Repo - http://update.datical.com/latest/thirdparty
  4. Select Datical DB ThirdParty Repo. You see a list of items available for installation.



  5. Select one or more of the following, then click next.

    • Subclipse

      This plug-in is for Subversion (SVN). If desired, you may open the Subclipse folder and uncheck Subclipse Integration for Mylyn 3.x. 

    • Team Explorer Everywhere

      This plug-in is for Microsoft Team Foundation Server (TFS)

  6. Install Details window. Inspect the list, then click Next. 
  7. Review License window. Click I accept, then click Finish. 
  8. When installation is complete, you are prompted to restart Datical DB. Click Restart Now.
    • If you do not restart Datical DB now, the changes take effect the next time you start Datical DB.
