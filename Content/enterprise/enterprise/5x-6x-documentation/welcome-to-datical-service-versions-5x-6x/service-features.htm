Datical Service Features

User Authentication

  • Datical Service uses a Keycloak service to integrate with your Active Directory/LDAP authentication service.
  • Users and groups are managed exclusively in Keycloak
    • Automatic - Federated integration imports your
    • Manual - Can set up groups and users manually for smaller-scale testing
  • Supports hierarchical groups

Role-based User Authorization

  • Predefined roles are provided.
  • Group assignment is preferred
  • User assignment is supported 

Datical Projects Management

  • Create project in the UI
  • Import projects created in Datical DB
  • Export projects from Datical Service to use with Datical DB

User interface

Deployment Management Console (DMC) is used for tracking status and performing manual forecasts and deployments.

  • Dashboard shows statistics for all project operations
  • Projects: create, import, export
  • Authorization Role management for groups and users

CLI for Datical Service Management

  • Provides monitoring and recovery operations

High Availability

  • Provides continuing availability in the event of a host failure
  • Three-node cluster required, one node per host
  • Recovery time is approximately 15 minutes if a host goes down
  • VM installations can be used with other VM-availability solutions

Disaster Recovery Support

  • Full system backups provide the basis for recovery.
  • All backup data is in one place for your use in your offsite-archiving processes
