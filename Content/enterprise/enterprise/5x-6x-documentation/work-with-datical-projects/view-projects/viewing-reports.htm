Viewing Reports

Reports are available in the reports tab at each level of a project. 

  • Project Reports
  • Pipeline Reports 
  • Step Reports

Report Type

  • Package - Results from packaging SQL files into changesets - the packaging operation is performed outside of Datical Service by running Deployment Packager from a command line or in an automation. 
  • Forecast - Results from testing a change against rules and simulating the change against the database. Warns if a SQL change cannot be forecast. 
  • Deploy - Results from a deploying changesets. It normally includes a forecast operation before deploying. 

More Files (menu)

Click the menu to view the available files. 

  • Output Log - Full log for an operation. When reporting problems to Datical Customer Support, attach this entire log to the support request. 
  • Deploy Script - (if specified in Deployment Options). SQL script of the change, generated from a deploy operation. 
  • Rollback Script -  (if specified in Deployment Options). Rollback script for the SQL script, generated from a deploy operation.
