Datical Active Operations

When operations run, they register with Datical Service to record their activity. The records are kept in a table in the Datical Service application database. 

Normally the record is removed automatically when the operation finishes. 

If an operation fails but is still shown here, use the Clear button to remove the record.

Note

The Clear button has no effect on the operation itself, just on its registration record. 

Normally you need to clear a record only if an operation failed unexpectedly. 

All operations register their activity:

  • Operations run from DMC
  • Operations run by the automation client (CLI), either directly by a user or through an automation. 

Operations may fail in ways that prevent the operation from finishing and updating its records. 

  • Network or system failure
  • Cancellation of an automation step that runs the operation
