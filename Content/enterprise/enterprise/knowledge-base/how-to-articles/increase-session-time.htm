How to increase your session time in Datical Service

By default the timeout are pretty short ant it will require that you log in again on fairly regular basis. The value of those tokens can be changed as need following the 

  1. Go to your server page http://daticalsvc.mycompany.com/auth
  2. Login into keycloak with the login and password you've created during the installation. 
  3. Go to ‘realm settings’ at the top:
  4. Go to Tokens:
  5. Adjust those values to fit your environment. More details can be obtained for each value by mousing over the question marks.
