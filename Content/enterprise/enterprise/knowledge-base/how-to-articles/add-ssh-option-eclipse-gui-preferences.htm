How To: Add SSH option in the Eclipse GUI/desktop client Preferences

After upgrading to later versions, you might have issues with the SSH feature being unavailable to generate the SSH key via the Eclipse GUI.

To return SSH in Preferences, follow these steps:

  1. In the GUI, select Help > Install New Software.

  2. In the Work with field of the Install dialog box, paste in https://download.eclipse.org/releases/2018-12

  3. Then hit the <enter> key, a menu should appear

  4. Expand open the Collaboration category

  5. Enable the checkbox to select Git integration for Eclipse, and then select Next to install it.

  6. Accept the terms and click Next

  7. When it finishes installing you should be prompted to restart

  8. Open File, then Preferences, then General, then Network Connections

  9. You should see a new SSH2 option
