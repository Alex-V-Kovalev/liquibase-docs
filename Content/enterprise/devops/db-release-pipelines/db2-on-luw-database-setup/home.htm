Install the CLPPlus Database Client

This guide works for both Linux and Windows users.

  1. Ensure db2 is installed by running the following command:
    user@1234 ~ $ clpplus -version
  2. If the CLPPlus version appears as depicted below, CLPPlus is installed already and you do not need to continue following these instructions.
    CLPPlus: Version 1.6
    OR
    If the CLPPlus version does not appear, continue with the steps below.
  3. Ensure that the installation directory is in the PATH.
  4. Locate and Install DB2 utility "IBM Data Server Runtime Client" from IBM Support's download page. 

      Linux Install:

    • Untar the download file: tar zxvf <tar_file>
    • Run setup:
      1. cd <untar_dir>
      2. ./db2setup (may need to update OS libraries by running "yum update")
      Windows Install:
      1. Double-click the downloaded file to open the install dialog.
      2. Select the I accept both the IBM and the non-IBM terms radio button and click Next.
      3. Select the program features you want installed and click Next.
      4. Set the IBM data server driver copy name and click Next.
      5. Confirm you are satisfied with the settings selected and click Install.
      6. Once the setup is complete, click Finish to exit the Setup wizard.
      7. Restart your machine for the configuration changes made to the IBM Data Server Driver Package to take effect.
  5. Verify db2 client setup
    • At the command prompt, run:
    • db2 ?

On success
Both CLPPlus and DB2 CLI tool are installed successfully. The version number of the db2 client will display when the clpplus -version command runs:
CLPPlus: Version 1.6

On error
The SQL10007N Message will appear:
-1390" could not be retrieved. Reason code: "3

  • Your db2 client is not setup correctly. To fix this error, run the db2icrt command as documented at https://www.ibm.com/support/knowledgecenter/en/SSEPGG_11.1.0/com.ibm.db2.luw.admin.cmd.doc/doc/r0002057.html
  • DB2DIR/instance/db2icrt -s ese -u < user> <db2 instance>
