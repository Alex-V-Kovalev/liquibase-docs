Relocate a Subversion Project

  1. Go to the Repository Explorer in
  2. Click the Deployment icon, then click Other.

  3. Select SVN Repository Exploring.

  4. Right-click on a project, then select Relocate.

  5. Click Next

  6. Update the New URL: field and click Finish

7. Confirm the change. The confirmation window shows you the active projects that are affected by relocation.
