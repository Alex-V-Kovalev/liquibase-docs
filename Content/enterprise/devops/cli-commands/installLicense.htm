installLicense

Install a license file to allow continued use of . Once a license is installed on a machine once, it does not typically need to be installed again.

USAGE

installLicense <path>

EXAMPLE

installLicense /home/datical/datdb.lic

OPTIONS

  • <path> – Specifies where to find the file that contains the <MadCap:variable name="General.DaticalDB" /> product license.

NOTES

If you need to obtain or renew a license please contact our Technical Support team.
