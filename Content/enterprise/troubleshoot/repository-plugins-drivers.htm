Unable to Read Repository for Plugins and Drivers

org.eclipse.equinox.p2.core.ProvisionException

After installing a new version of DaticalDB, some users may experience this error from either the GUI or the CLI when attempting to update drivers or install plugins. This may be caused by using a 32-bit DaticalDB after having previously used a 64-bit DaticalDB, or vice-versa.



To resolve: Remove the version that caused the error, then install the correct version.
