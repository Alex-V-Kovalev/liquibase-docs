Unable to Read Repository for Plugins and Drivers

org.eclipse.equinox.p2.core.ProvisionException

After installing a new version of , some users may experience this error from either the GUI or the CLI when attempting to update drivers or install plugins. This may be caused by using a 32-bit after having previously used a 64-bit , or vice-versa.


Resolution:
Remove the version that caused the error, then install the correct version.
