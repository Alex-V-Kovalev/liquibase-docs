Feature Support

  • SSL connections through Oracle Wallet - Supported.

    • See SSL Connection to an Oracle Database Through Oracle Wallet.

    • See also AWS documentation on using SSL - https://docs.aws.amazon.com/AmazonRDS/latest/UserGuide/Appendix.Oracle.Options.SSL.html

  • Pluggable Databases (PDBs) - AWS does not support Pluggable databases on Oracle RDS 

  • Oracle Flashback database - AWS does not support Flashback database on Oracle RDS
