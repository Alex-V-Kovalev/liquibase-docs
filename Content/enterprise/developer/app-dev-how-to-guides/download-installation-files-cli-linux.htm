How To: Downloading Installation Files Using the Command Line on a Unix/Linux Machine

Overview

In order to make it easier to download and DMC installation files for Unix/Linux, has created shell scripts that uses a 3 step process.

Prerequisites

The script will first ensure that you have installed two command line utilities required.

  1. The first is curl, a tool for interacting with websites from the command line. This is typically pre-installed on most Linux distributions.

  2. The second utility is jq, a tool for parsing JSON-formatted data. This is less likely to be installed. If either tool is not installed, the script will provide instructions for installing it.

  3. When both utilities are installed, running the script will then prompt you for your username and password for software.datical.com. It will then begin downloading the installation files, providing progress reports during the process.

Instructions

  1. From the Linux command line, download the script:

    1. For run: wget "https://drive.google.com/uc?export=download&id=1hQh49saWbwq7CNQMdyNbQcnr_qH1Bi9e" -O get-datical-db-linux.sh

    2. For DMC run: wget "https://drive.google.com/uc?export=download&id=1rJS_4qtfoDotw-tBKJg6ScDRKtm4yhxp" -O get-dmc.sh

  2. Run permissions on the script to make it executable: chmod +x get-datical-db-linux.sh or chmod +x get-dmc.sh

  3. Download the installer.

    1. Execute the command and specify the version of Datical version you want to install: ./get-datical-db-linux.sh <version #> or ./get-dmc.sh <version #>

    2. Here is a real time example: ./get-datical-db-linux.sh 7.5.6684

    3. Here is the link to our Datical Release Notes for the specific versions we currently have available on our download site.

    4. Please make sure you have access to our download site as you will be prompted for software.datical.com username and password.

Once the installer is fully downloaded please continue with our https://datical-cs.atlassian.net/wiki/spaces/DDOC to complete installation.
