Rename a Project Outside of Source Control

If a project is not under source control, use the following method to rename it.

  1. In a Datical project directory, modify the .project & datical.project files with the new name.



  2. In the GUI, right-click the project, then select Delete. When asked to confirm, DO NOT select Delete project contents on disk (cannot be undone)



  3. Move the project folder to a TEMP folder outside of the project directory and rename the folder to match the new project name

  4. Choose File > Import to import the project from the TEMP directory
