How To: Install 2019.2.2 or 2019.2.3 GUI

Step-by-step guide

Please note: These instructions are ONLY for users that are running 4.x or 5.x versions and are now upgrading to versions 2019.2.2 or 2019.2.3.

Download the required installation file and the drivers file "compositeRepo.zip" (we are using Windows 64 bit operating system in this example)

Go to https://software.datical.com on the left side, click on:  → FOLDERS → COMMON → Datical_DB_Software → Datical_DB_2019.2.3.6054 and download the two files: DaticalDB-win-x86_64-2019.2.3.6054.exe and DaticalDBCompositeRepo-2019.2.3.zip

Backing up the workspace .metadata file

1. Go to your workspace directory typically located at C:\Users\<username>\datical and create a new sub-folder called "backup".

2. Move the .metadata folder to the backup folder.

Do a fresh install

  1. Double click the DaticalDB-win-x86_64-2019.2.3.6054.exe file and do a fresh install.
  2. After finishing installing , launch DaticalDB GUI.
  3. At this point, the GUI will not have any projects appearing. This is ok.

4. Click on Help → Install New Software → Add button → Archive Button → and navigate to your DaticalDBCompositeRepo-2019.2.3.zip file → click "Open" → add button

 

5. Under "Available Software" choose the following:

6. Click next → next → click "I accept the terms of the license agreement (if you agree) → Finish → In the Security Warning window (read the warning) click the "Install anyway" button → click Restart Now button.

7. After the GUI appears again, close it.

8. Go back to the backup folder you have previously created at C:\Users\<username>\datical\backup

9. Copy the .metadata folder and paste it at the workspace folder C:\Users\<username>\datical

Windows window will show up and ask you if you want to Replace the files in the destination. Click "Replace the files in the destination".

10. You should now be able to see all of your projects on the GUI.

Note that following these steps in order is especially important with versions 2019.2.2 and 2019.2.3. (The process is somewhat easier with 2019.2.4 and higher.)
