How To: Setup Network Connections with a Proxy Sever in the UI

Step-by-step guide

  1. Open the Datical UI and Click → File → Preferences
  3. Click "Network Connections"
  4. Under "Network Connections" windows switch the Active provider drop-down menu to "Manual"
  6. Choose your Proxy connection from the "Proxy entries" section and click the "Edit..." button (in this example it is "HTTP")
  8. In the Edit Proxy Entry window fill in your connection details and user and password if applicable and click the OK button
