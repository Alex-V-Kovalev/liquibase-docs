Using a Third-Party Enterprise Password Vault with

Many of our customers prefer to use a third-party password vault such as Cyber Ark to manage credentials in their CI/CD pipeline.  employs a loose-integration with these products through the use of environment variables as follows.

Step-by-step guide


  1. Retrieve your credentials from your enterprise password vault
    1. Cyber Ark REST API Demo: https://www.cyberark.com/resource/cyberark-pas-version10-user-interface-rest-api-demo/
  2. Set the environment variables as appropriate with credentials retrieved in step 1 above
    1. Environment variables used can be found in: Using Environment Variables to Provide Runtime Credentials 
  3. Call CLI "hammer" commands - these commands consume the credentials specified in step 2 above
    1. Example CLI commands can be found in: CLI Commands
