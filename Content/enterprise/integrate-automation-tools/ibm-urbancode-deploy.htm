IBM UrbanCode Deploy Integration

Download the DaticalDB4UrbanCode Plug-in

  1. Download the DaticalDB4UrbanCode plugin. .
  2. Go to the following URL: https://github.com/IBM-UrbanCode/DaticalDB-UCD-UCB/releases
  3. In the latest release, click the .ZIP file (DaticalDB-UCD-UCB-<version>) to download it.

Install the Plug-in 

  1. In UrbanCode Build, navigate to Settings > Automatic Plugins.
  2. Click Load Plugin.
  3. Click Choose File.
  4. Select the downloaded DaticalDB4UrbanCode zip file.
  5. Click Submit.

Set Properties

  1. Click Resources.
  2. Go to an agent resource.



  3. Click View Agent to list the agent properties.
  4. Go to Configuration > Agent Properties to add the following properties.
    • daticalDBCmd
    • daticalDBDriversDir


Add Steps to UrbanCode Deploy Processes

  1. Go to a Process. 
  2. From Available Plugin Steps, choose a Datical step and drag it to the process.



  3. Configure the step for your environment. 
    • The daticalDBCmd and daticalDBDriversDir options are already populated with the property you created.
    • If necessary, enter the path of your Project Directory relative to your project top level folder. If you checked your Project Directory into the root, use "." (period).
  4. Add further steps as needed.
