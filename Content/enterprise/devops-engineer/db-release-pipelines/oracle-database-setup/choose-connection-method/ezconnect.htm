Use an EZCONNECT Connection to an Oracle Database

Use this type of connection to provide connection information directly in the DbDef.

Fill in the following properties when you create the DbDef in the project.

DbDef Properties

The following properties are required to create the connection.

  • Connection Type- Choose BASIC

  • Hostname - Hostname or IP address for the database.

  • Port - 1521 by default

Fill in either Service Name or SSID.

  • Service Name - Oracle: Service Name (orcl is the default)

  • SSID - Oracle: Service Identifier

See Configuring Step Settings (DbDefs)#OracleDatabaseStepSettings for the complete list of Oracle DbDef properties.
